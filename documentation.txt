# Contact Web App - Development Documentation

## Overview

This document outlines the requirements, validation rules, and expected behavior for the Contact Web App. The goal is to ensure all team members are aligned on the development process.

---

## **Tech Stack**

- **Backend:** Flask (Python) with Flask-PyMongo
- **Database:** MongoDB
- **Frontend:** HTML, CSS, JavaScript (with real-time validation)
- **UI Framework (Optional):** Bootstrap or React

---

## **Database Schema**

Each contact will be stored with the following fields:

```json
{
  "name": "John Doe",
  "prefix": "+91", // Only if manually selected
  "phone": "9876543210", // Must be exactly 10 digits
  "email": "john.doe@example.com",
  "address": "123 Street, City, Country"
}
```
